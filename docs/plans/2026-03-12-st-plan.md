## 3. Entry Criteria

- `python3 scripts/check_st_readiness.py feature-list.json` returns `READY`
- SRS and Design documents exist
- Required remote configuration exists
- Local build tree is available

## 4. Exit Criteria

- Applicable regression/integration/E2E/performance/security/compatibility/exploratory checks have evidence
- `NFR-001`, `NFR-002`, and `NFR-003` have measured evidence
- No Critical or Major defects remain open
- RTM coverage is 100% for `FR/NFR/IFR`

## 5. Risk-Based Prioritization

1. Critical path: benchmark matrix, recall threshold, reproducibility bundle
2. Cross-feature boundaries: CLI -> loader -> builder/search -> metrics -> result files
3. Remote compatibility: remote readiness and benchmark host smoke
4. Error handling: invalid dataset and missing-path diagnostics
5. Residual risk: coverage tooling and mutation tooling availability in the current ST environment

## 6. Planned System Tests

### Regression
- Local `cmake --build build -j`
- Local `ctest --test-dir build --output-on-failure`
- Coverage attempt with `gcovr` in a coverage build directory if tooling is available

### Integration
- Validate feature 11, 12, 13 artifacts together through `results/repro_manifest.json`
- Validate result JSON schema and metric availability
- Validate remote x86 smoke via `check-env.sh` and remote `ctest`

### E2E
- Local synthetic PiPNN CLI run with `PIPNN_PROFILE=1` and `PIPNN_ECHO_CONFIG=1`
- Local synthetic HNSW CLI run
- Repro manifest validator run

### Performance
- Use recorded benchmark JSON files and report for:
  - `100k/100`
  - `200k/100`
  - `500k/100`
- Compare PiPNN and HNSW build times
- Verify `beam=384` threshold config

### Security
- Manual audit for unsafe shell execution, insecure file handling, and secrets in repo
- Lightweight grep-based scan for command execution primitives and embedded secrets

### Compatibility
- Local macOS build/test pass
- Remote x86 readiness check
- Remote x86 `ctest` smoke pass

### Exploratory
- Unsupported dataset argument
- Missing file path error
- Profile/config echo toggles on a local synthetic run
